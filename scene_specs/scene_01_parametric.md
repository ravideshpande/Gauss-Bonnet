# Scene 01 — Parametric Curves

## GOAL
Show that curves should be thought of as moving points, not functions.

The viewer should feel:
"A curve is a path traced by motion."

## MATH
- y = f(x) vs parametric form
- x = x(t), y = y(t)
- p(t)

## VISUAL SEQUENCE
1. Show a circle and briefly illustrate that it cannot be represented as y = f(x)
2. Introduce a moving point tracing the curve
3. Display x = x(t), y = y(t)
4. Collapse into p(t)
5. Briefly extend to 3D: p(t) = (x(t), y(t), z(t))

## STYLE
- Smooth motion
- Minimal text
- Strong emphasis on moving point

## RUNTIME
~45 seconds
